## 💾 Featured Projects

### 🌌 Celestial Bodies Database

- **Focus:** PostgreSQL, Schema Design, Relational Mapping.
- **Description:** Designed a universe database containing planets, moons, stars, and galaxies with complex foreign key relationships.
- **File:** [`/celestial_bodies/universe.sql`](./celestial_bodies/universe.sql)

### ⚽ World Cup Database

- **Focus:** Bash Scripting, PostgreSQL, Data Migration.
- **Description:** Created a Bash script to automate the migration of CSV data into a normalized SQL database and queried the results.
- **Files:** [`/worldcup`](./worldcup)

### ✂️ Salon Appointment Scheduler

- **Focus:** Bash Scripting, PostgreSQL, Database Automation.
- **Description:** Developed an interactive appointment system using a Bash interface to manage relational data and automate customer scheduling logic via SQL.
- **File:** [`/salon_appointment_scheduler`](./salon_appointment_scheduler)

### 🧪 Periodic Table Database

- **Focus:** SQL, Bash Scripting, Data Cleaning, Relational Databases.
- **Description:** Built a robust database of chemical elements by importing data from CSV files, cleaning inconsistencies via Bash and SQL, and creating a script to provide detailed element information through terminal queries.
- **File:** [`/periodic_table_database`](./periodic_table_database/)
